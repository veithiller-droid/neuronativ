Datenerfassung auf dieser Website

Diese Website wird auf GitHub Pages gehostet. Es werden **keine personenbezogenen Daten** serverseitig gespeichert oder verarbeitet.

Der Selbsttest verwendet ausschließlich **localStorage** Ihres Browsers, um Ihre Eingaben und Ergebnisse temporär zu speichern. Diese Daten bleiben **ausschließlich auf Ihrem Gerät** und werden **nicht** an Server oder Dritte übermittelt.

Es werden **keine Cookies**, **kein Tracking** (z. B. Google Analytics) und **keine Server-Logs** verwendet.

Ihre Rechte

Da keine personenbezogenen Daten serverseitig gespeichert werden, entfallen Anfragen auf Auskunft, Löschung oder Berichtigung. Lokale Daten können Sie jederzeit selbst löschen, indem Sie den Browser-Cache leeren.

Bei Fragen zum Datenschutz erreichen Sie mich unter info@neuronativ.de.
